# CDFLOW

All code need to generate CDFLOW is included within the folder labeled “Main”. 
Steps to generate go through downloading data, manipulating the data, estimating CO2, then some QA/QC. 
CDFLOW data generation follows an iterative process in which the scripts are labeled with the order they are intended to be run. 
Slight modifications will need to be done in order to run on your machine and for your specific purposes. 
However, we have provided comments throughout the code to guide you.
